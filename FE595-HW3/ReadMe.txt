Usage:


1. Download all the files including two folders 'File Name' and 'Merged data frame'.

2. Put your own csv files and txt files into 'File Name' folder.

2. Run Script 1, it will output a csv with all company names and company purposes in the folder 'Merged data frame'.

3. Run Script2, sentiment score will be calculated and it will return you the best idea and worst idea of the company.

4. Run Script3, it will tell you the most common words in the company description.
